Added Discovery Pipeline instructions securely executing context-aware targeted edits to the `SYSTEM_PROMPT`.
- **GitHub Copilot**: Fixed an unhandled `KeyError` crash in `dispatch_tool` inside `llm_code_agent.py` by wrapping tool execution in a `try...except KeyError` block. Missing LLM tool arguments (like omitting `path` when calling `edit_file`) are now returned as an error response string rather than immediately halting execution. This enables the agent to autonomously detect the error, observe the missing property, and self-heal on the next iterative step without terminating the entire operational shell entirely.

Re-established the mandated `browser_observe` protocol upon interaction failures.
